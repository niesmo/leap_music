Wad.js Demo Song

Wad is a Javascript library for manipulating audio using the new HTML5 Web Audio API. It greatly simplifies the process of creating, playing, and manipulating audio, either for real-time playback, or at scheduled intervals.


Press the "Play!" button to start playing a demo song. Hover over any of the instrument buttons at the top of the screen to read a brief description of how that instrument was synthesized in this song. Click any of the buttons to mute or unmute that particular instrument. If you mute or unmute an instrument while the song is playing, it may take several seconds for the change to take effect.
